O que é JavaScript?

JavaScript é a linguagem de programação da Web. É usada para criar interatividade em páginas HTML.

Como inserir JavaScript

  • Embutido: <button onclick="alert('Olá!')">Clique</button>
  • Interno: dentro da tag <script>
  • Externo: <script src="script.js"></script>

Variáveis em JavaScript

var nome = "João";
let idade = 25;
const PI = 3.14;

Teoria: var, let e const

  • var – Escopo global ou de função. Pode ser redeclarada e atualizada. Usada em versões antigas do JavaScript.
  • let – Escopo de bloco. Pode ser atualizada, mas não pode ser redeclarada no mesmo escopo. É a forma recomendada.
  • const – Escopo de bloco. Não pode ser atualizada nem redeclarada. Usada para valores constantes.

Dica: Sempre prefira let e const para evitar erros comuns em código moderno.

Mais sobre const

  • const cria uma variável com valor constante, ou seja, não pode ser reatribuída.
  • Ideal para valores fixos, como configurações, nomes, URLs, etc.
  • ⚠️ Importante: se for um array ou objeto, o conteúdo pode ser modificado, mas a referência não!
// Exemplo com string
const nome = "Maria";
nome = "João"; // ❌ Erro! Não pode mudar

// Exemplo com array
const numeros = [1, 2, 3];
numeros.push(4); // ✅ Pode mudar o conteúdo

// Exemplo com objeto
const config = { tema: "claro" };
config.tema = "escuro"; // ✅ Pode alterar propriedade

Dica: use const como padrão e apenas let quando precisar mudar o valor.

Mais sobre let

  • let é usado para declarar variáveis com escopo de bloco.
  • Ao contrário de var, o let respeita blocos como if, for e while.
  • Pode ser atualizado, mas não pode ser redeclarado no mesmo escopo.
// Exemplo de escopo com let
let idade = 18;
if (true) {
  let idade = 25;
  console.log(idade); // 25 (escopo interno)
}
console.log(idade); // 18 (escopo externo)

Resumo: Use let quando o valor precisar mudar, mas sempre prefira const quando possível.

Aprofundando: Funções em JavaScript

  • Funções são blocos reutilizáveis de código que executam tarefas específicas.
  • Podem ter parâmetros (entradas) e retornar valores (saídas).
  • Existem funções declaradas, expressões de função e arrow functions.
// Declarada
function saudacao(nome) {
  return "Olá, " + nome;
}

// Expressão
const soma = function(a, b) {
  return a + b;
};

// Arrow function
const dobro = x => x * 2;

Aprofundando: Manipulação do DOM

  • DOM é a representação da estrutura da página HTML em objetos JavaScript.
  • Permite acessar e modificar conteúdo, estrutura e estilo de forma dinâmica.
  • Principais métodos:
document.getElementById("id");
document.querySelector(".classe");
document.querySelectorAll("div");

Você pode alterar textos, classes, atributos e estilos diretamente.

Aprofundando: Eventos

  • Eventos permitem responder a ações do usuário: cliques, teclas, movimentos etc.
  • Uso comum: onclick, onmouseover, onkeyup.
  • Boas práticas: usar addEventListener.
// Exemplo moderno:
document.getElementById("botao").addEventListener("click", function() {
  alert("Você clicou!");
});
